# AMC 8 Timer Tracker

A sleek, high-contrast light theme web application designed to help students practice for the AMC 8 competition. The app tracks time spent on each individual problem and provides an automated results summary based on the official Art of Problem Solving Wiki answer keys.

## Features

- **Dynamic Problem Loading**: Paste any AMC 8 Problems URL from the [Art of Problem Solving Wiki](https://artofproblemsolving.com/wiki/index.php?title=AMC_8_Problems) to begin.
- **Auto-Advance Interaction**: Selecting an answer choice automatically saves your response, stops the individual problem timer, and scrolls to the next problem.
- **Change Answer Support**: Return to any completed problem to modify your answer; the individual timer will resume exactly where it left off.
- **Sticky Countdown**: A global 40-minute countdown clock remains visible as you scroll through the problem set.
- **Visual Analytics**: After the test, view a comprehensive breakdown of your performance, including correct/wrong/skipped counts and a per-problem time analysis.
- **Light Theme Optimization**: High-contrast design ensures that LaTeX math images and complex equations are perfectly legible.

## Technologies Used

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Premium, responsive styling with a focus on high legibility.
- **Vanilla JavaScript**: Core application logic, MediaWiki API integration, and timer systems.

## Getting Started

1.  Clone this repository.
2.  Open `index.html` in any modern web browser.
3.  Paste a link like `https://artofproblemsolving.com/wiki/index.php?title=2020_AMC_8_Problems`.
4.  Start your practice session!
